Power BI Bağlantı Rehberi

  1. Power BI Desktop → Get Data → Web
  2. URL alanına yukarıdaki "KPI Ölçümleri" CSV bağlantısını yapıştır
  3. Authentication → Anonymous (Kokpitim oturumu açık olmalı) veya Basic Auth ile API token
  4. Veri yüklendikten sonra Power Query Editor'da gerekli dönüşümleri yap
  5. Direct Query desteklenmiyor — Import mode kullan (yenileme zamanlı)

Tableau Bağlantı Rehberi

  1. Tableau Desktop → Connect → To a Server / Text File
  2. URL ile bağlanmak için Tableau Web Data Connector (WDC) gerekir
  3. Veya CSV'yi indir → Tableau'da Text File kaynağı olarak aç
  4. Yenileme manuel; otomatik refresh için Tableau Server gereklidir

Excel / Google Sheets

  1. Excel: Data → From Web → URL'yi yapıştır
  2. Google Sheets: =IMPORTDATA("URL")
  3. Veri otomatik formatlanır, sütun başlıkları algılanır
MVP Notu: Bu MVP CSV/JSON endpoint'leri. Tam OData v4 desteği (canlı yenileme, filtre, hiyerarşi navigasyon) için ek geliştirme gerekir (60h tahmini).
